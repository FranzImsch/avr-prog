# AVR-Prog
Basic Programmer for AVR-Chips. It's based off the [Sparkfun Tiny AVR Programmer](https://github.com/sparkfun/Tiny-AVR-Programmer).

![Rendering](rendering.png)

Features are:
- USB-C connector,
- 5V or 3V3 logic level,
- Tag-Connect connector.
